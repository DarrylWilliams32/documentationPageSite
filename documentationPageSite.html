Introduction

HTML (Hyper Text Markup Language) is the primary building block of nearly all websites on the internet. It is this reason that it's imparitive that you learn how to use it. For example, content could be structured within a set of paragraphs, a list of bulleted points, or using images and data tables.

Setting up HTML document

Before you can start writing on your html document, you must first let your computer know that this is the document type you are looking to edit. To do this, you must create an html document in your text editor of choice- we wont cover how to do that here. Secondly, you must put the appropriate heading on the document that establishes the 'Document type'. This can be achieved with the following:

<!DOCTYPE html>

Here, the <!DOCTYPE html> tag is used to tell the computer that this is an html document. This is the first thing that should be written on your document, and should be written on the first line.

Once you've put the appropriate heading, you can now edit this document as html. There are some additional tags that are both useful, and industry standard to make your document more readable, accessible, and styleable.

  • <html> : This tag should encapsulate the entirety of the document. Best practice is to place it directly below your <!DOCTYPE> tag and then at the end of the document.

  • <head> : This tag should encapsulate the metadata of the document. This includes the title, links to stylesheets, and other information that is not displayed on the page.

  • <body> : This tag should encapsulate the content of the document. This includes the text, images, and other information that is displayed on the page.

  • <title> : This tag should encapsulate the title of the document. This is the text that will be displayed on the tab of the browser.

  • <link> : This tag should be used to link to stylesheets. This is the best way to style your document, and is the industry standard.

Write your first message

HTML serves as the framework for many sites an applications, so lets cover the basics first.

Let's learn how to print your first message to the world!

<p>Hello World</p>

Here, the <p> tag is used to create a paragraph to that will be printed to the screen. These are a great way to display messages and information on your website. These are best fit for when you have a paragraph you want to display, but shouldn't be used as headers or titles - which we will cover later.

Add a header

Headers are a great way to display titles and subtitles on your website. Titles can be a great way to let your user know what the following paragraph or section is going to be about.

Let's learn how to add a header to your website!

<h1>This is a header</h1>

Here, the <h1> tag is used to create a header. This is the largest header you can create, and should be used for titles and subtitles. If you want to create a smaller header, you can use the <h2> tag, and so on.

Organizing information

There will be times that you need to display information to a user in the form of a table or graph - we will cover the former here.
Tables are a simple and effective way to organize information for the user to read. These can be used to create:

  • calendars
  • charts
  • graphs
  • and more!

There is a variety of HTML tags that you'll use to create a table and they are: <table>, <h1>

  • <table>: table is the first tag of a table and it must encapsulate all the information that you wish to display within.
  • <tr>: table row is used to create rows within your table
  • <td>: table data is a versatile tag in that it can be used to add information to your table and it can be used to create columns within the table. There is no specific tag to make columns and so the table designer must use parameters within the <td> tag to acheive this.

Adding media

Now that you've leared how to display all kinds of text based information, it's time to add more dynamic content. This is where media comes in. There are various kinds of media that you can add to your site such as: images, audio, video, etc

Media tags

<img>: the image tag is used to add images to your site. If you are referencing an image that is online outside of your project, you must reference it's destination with href. If you are using an image that is located within your project directory, then you will use src to reference is location.

Using external images
